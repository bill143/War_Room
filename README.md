# ⚔️ The War Room

AI-Powered WebRTC Meeting Intelligence Platform

## Overview
Real-time meeting transcription, speaker diarization, and AI-powered intelligence extraction using Deepgram Nova-2 and GPT-4o.

## Setup
1. Copy `.env.example` to `.env` and add your API keys
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:8765`
